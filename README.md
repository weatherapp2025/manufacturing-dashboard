# manufacturing-dashboard
Developed a Manufacturing Dashboard project developed using Python, Pandas, NumPy, Matplotlib and Seaborn for production analysis and KPI visualization.

# Skills Used
- Python
- Pandas
- Numpy
- Matplotlib
- Seaborn
- Data Analysis
- Data Visualization

# Features
- Production Analysis
- KPI Monitoring
- Shift Wise Production Tracking
- Product Category Analysis
- Dashboard Reporting
